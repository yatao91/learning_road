## GOPATH设置

> GOPATH环境变量指定了工作目录的位置。
>
> 从go 1.8开始，GOPATH环境变量有一个默认值，在Linux操作系统中是`$HOME/go`
>
> ```
> # Bash
> vim ~/.bash_profile
> 
> # 最后一行后面增加
> export GOPATH=$HOME/go
> 
> # 更新环境变量
> source ~/.bash_profile
> ```
>
> ```
> # Zsh
> vim ~/.zshrc
> 
> # 最后一行后面增加
> export GOPATH=$HOME/go
> 
> # 更新环境变量
> source ~/.zshrc
> ```
