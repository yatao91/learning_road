##### 1. 安装

> ```
> pip install coverage
> 
> 安装完成后使用以下命令检查:
> ~ coverage --version
> Coverage.py, version 4.3.2 with C extension
> Documentation at https://coverage.readthedocs.io
> ```
>

##### 2. 命令行用法

> 1. 基本命令
>
>    ```
>    run: 运行Python程序并收集执行数据
>    report: 报告覆盖率结果
>    html: 生成HTML报告
>    xml: 生成XML格式的报告
>    annotate: 使用测试结果注释原文件
>    erase: 擦除之前收集的覆盖数据
>    combine: 将需要覆盖报告文件整合在一起
>    debug: 获取诊断信息
>    ```
>
> 2. 帮助命令的使用
>
>    ```
>    coverage help
>    coverage help run
>    coverage run --help
>    ```
>
> 3. 基本执行用法
>
>    ```
>    $ coverage run my_program.py arg1 arg2
>    blah blah ..your program's output.. blah blah
>    ```
>
> 4. 一般用法
>
>    ```
>    1. 对多个测试用例分别生成覆盖率报告:
>    coverage run -p test_*.py
>    
>    2. 合并多个测试用例覆盖率报告:
>    coverage combine
>    
>    3. 查看测试用例未覆盖的代码行(-m:表示在输出报告中显示Missing列):
>    coverage report -m
>    
>    4. 测试覆盖率报告中不显示测试用例模块:
>    coverage report --omit="test_*.py"
>    
>    5. 也可以在执行的时候指定过滤,忽略符合test_*.py模式的文件:
>    coverage run --omit="test_*.py" test_user.py
>    ```
